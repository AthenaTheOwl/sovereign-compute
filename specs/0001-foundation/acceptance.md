# Spec 0001 — Foundation acceptance

## What "v0 done" means

Spec 0001 is closed when:

1. The three PRs in `tasks.md` are merged.
2. `uv pip install -e .[dev]` succeeds on a fresh venv.
3. `python -m sovereign_compute --help` prints the CLI surface.
4. The five seed programs (UAE G42, Saudi HUMAIN, EU AI Factories,
   IndiaAI, UK AISI) all load, validate, and score end to end.
5. `reports/2026-Q3-scorecard.md` exists, was generated by the
   pipeline, and passes voice_lint plus the no-advocacy check.
6. All local gates pass.

## Commands to run

```bash
uv pip install -e .[dev]
python -m sovereign_compute --help

uv run pytest
python scripts/voice_lint.py
python scripts/validate_schemas.py

python -m sovereign_compute scorecard \
  --programs programs/ \
  --silicon-data data/silicon_allocations_2026q2.json \
  --grid-data data/grid_capacity_by_country.json \
  --water-data data/water_basin_by_site.json \
  --talent-data data/fte_supply_by_country.json \
  --out reports/2026-Q3-scorecard.md

python eval/sanity_bounds.py reports/2026-Q3-scorecard.json
```

## Gates

| Gate | Source | Blocks merge when |
|---|---|---|
| pytest | `tests/` | any test fails |
| voice_lint | `scripts/voice_lint.py` | banned term or advocacy verb in `reports/` |
| schemas | `scripts/validate_schemas.py` | any program or scorecard fails |
| sanity bounds | `eval/sanity_bounds.py` | projected > announced, missing axis, or fewer than 3 citations |

## What v0 explicitly does NOT include

- Recommendations on program design or budget reallocation.
- Forecasts of geopolitical events.
- An interactive scorecard frontend.
- Hosted SaaS.
